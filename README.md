# Vitalii Shevtsov - Test assignment

- React + TypeScript + Vite

Deployed app: [https://vitaliyshevtsov.github.io/](https://vitaliyshevtsov.github.io/)

## Development Setup

### Run the project

Install packages:

```
npm install
```

Run the application:

```
npm run dev
```

###

Run tests:

```
npm run tests
```

Run test coverage:

```
npm run coverage
```
